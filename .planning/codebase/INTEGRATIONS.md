# Integrations

## External Services
- **Google Maps**: Embedded via iframe in `Contact.jsx`.
- **Transparent Textures**: Background pattern loaded from `https://www.transparenttextures.com/patterns/p6.png` in `index.css`.

## Internal Data
- **Services Data**: Managed through `src/data/services.js`.
- **Sub-Service Details**: Managed through `src/data/subServiceDetails.js`.
- **Service Sections**: Managed through `src/data/serviceSections.js`.

## Forms & Lead Capture
- **Contact Form**: Present in `Contact.jsx`, currently uses local state/prevent default. No backend integration (e.g., Formspree, SendGrid) detected yet.

## APIs
- No direct API consumption (REST/GraphQL) detected in the frontend code. The app is currently a high-fidelity static site powered by local data.
